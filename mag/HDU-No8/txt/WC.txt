================================================
Converts Word 2,5,6,7,8 files to text, HTML or AmigaGuide. 
Author:   docent@union.org.pl (Peter Drapich)
================================================

Word Converter is  a  useful  tool  for  everybody,  who is 
looking for the possibility  to read documents, created with 
Word - a famous word processor by  Microsoft,  available  on  
Wintel  and  Mac  platforms.  Thanks to Word Converter,  you 
are able to convert documents created with Word for Windows
version  2,  6,  7 and 8 and Word 4, 5 for Mac to plain text, 
AmigaGuide or HTML  document,  or even import them directly 
into PageStream 3 or CygnusEd text editor through additional 
ARexx scripts.

This is the updated evaluation version 0.9.4 of Word Converter.

Key Features

- Simple,  intuitive way of converting Word documents.
- Converts documents, created with Word for Windows
- Converts documents, created with Word 4, 5 for Mac.
- Converts documents, created with Word for DOS (hopefully).
- Docs can be converted to ASCII, HTML and AmigaGuide.
- Works as a command line tool too.
- Additional  ARexx  scripts  allow easy import of Word 
document directly into PageStream 3 or CygnusEd text editor.
- Context help (requires Kickstart 3.0)
- Quite stable.
