Please note the following changes/additions which are valid for 
GlobalTrash version 1.3 and couldn't be implemented in the 
documentation for this version.

Installation
If you install GlobalTrash by using the installerskript please 
ensure that you have the latest version of the Installer (current 
version 43.3) installed in the current path - otherwise the 
installation may not be finnished correctly or won't start.

How to register
In addition to the mentioned possibilities to register, there is one 
more possibility offered: you can register online on the 
support-pages (WWW - see Updates-section in the guide). Just 
fill out all the neccessary field in the form and send it.
