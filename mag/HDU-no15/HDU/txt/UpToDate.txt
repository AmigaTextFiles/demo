Requires : OS 2.0+

INTRODUCTION
This program calculates the difference in days between two 
dates, the weekday of a date, the date of Easter Sunday and 
displays a calendar of any month. Calculations a made by 
"date.library" (Author: Kai Hofmann - many thanks for this 
library!!)

INSTALLATION
There is no installer script yet (sorry...). Copy
-the main program wherever you want
-"date.library" in the libs: directory
-"date.catalog" in the "locale:catalogs/..." directory
(if you want to use German language instead of English).

USAGE
