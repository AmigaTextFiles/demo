===============================================
The Amiga Hard Disk Guide
Author : P.J.Hutchison@hud.ac.uk (Peter Hutchison)
===============================================
This guide is for new Amiga Users with hard disks. It explains 
how to install, partition, and install Workbench. It also gives hints 
on how to manage and look after your hard disk and and some 
troubleshooting tips.

Now includes:
* List of SCSI and IDE controllers for HDs
* How to install a new/alternative file system
* Details on patches for >4Gb drives
* Updated list of useful programs
