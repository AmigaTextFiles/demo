English
This is a free copy of the Gateway-5-BBS-StarDate-Clock-Utility.
Gateway-5 is a german only BBS-System. Because this, the 
internal help of StarDate is in german. 

Here the english translation:
? * show release-information and used tools for create
   this program.
-h * this help, but in german
-d<digits> * number of digits after decimalpoint
-t<hours> * timezone (-12 to 12 hours from greenwich)
-o<file> * file for output, don't print to cli

Requires : Amiga
The are version's for all different CPU-types from 68000 up to
powerpc (PowerUp).

Install : Copy the File StarDate in any directory you like and start 
it from CLI.
