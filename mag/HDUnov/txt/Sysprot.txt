Author:   dallsopp@enterprise.net (Daniel Allsopp)
============================================

SysProt'97 is a simple system to allow you to have different
users for your workbench. Features included with this version
are :

- Unlimited number of users;
- Users each have their own home directories;
- Sound sample attached to events;
- Amiga DOS script execution;
- Different settings for each users' Workbench;
- Simple mail messages;
- Time restrictions for loggin in;
- Multiple passwords;
- Password encryption;
- Logging of events;

This version of SysProt requires a 640x512 screen mode 
or equivilent and a hard disk drive.
