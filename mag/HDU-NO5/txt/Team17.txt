==========================================
 HD Installers for Team 17 games
 Author: jffabre@ensica.fr (Jean-François Fabre)
==========================================
This package allows to install many games from Team17:

Arcade Pool (no quit, no JST, no plain A1200)
All Terrain Racing
BodyBlows Galactic ECS
BodyBlows Galactic AGA  (no plain A1200)
Overdrive
Project X
SuperFrog

It has been completely updated. Most of the loaders now use 
JOTDStartup and run on standard A1200. Evaluation version 
v0.5c is included.
