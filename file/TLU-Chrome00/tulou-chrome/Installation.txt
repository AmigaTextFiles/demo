How to install it:

Download ALL files from Aminet:
TLU-Chrome00.lha --> TLU-Chrome10.lha

Extract all files in the same directory on your HD (all files will be placed under a directory called Tulou-Chrome). Note! You must have at least 250MB free.

How to run it:

The best way is to boot with no-startup. Go to the directory called Tulou-Chrome and write:

execute start


That´s it! Enjoy the show! :)
