
BURNING CHROME

This demo should run on every Amiga with 4MB Fastmem,
but 68030/50Mhz is recommended


To install this demo simply copy all files in one directory
and start it...

You can also run it from disk. In this case copy all files
except "BurningChrome.dat" to one disk and "BurningChrome.dat"
to a second disk.
The name of the second disk must be "BC2"
