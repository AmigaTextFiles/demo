Manipulations is a AGA/HD only demo that required 1,8 Mb of free memory.
If you have Fast-mem or better CPU the demo will be more admirable.
To install on HD just clik on the Install icon. The 'MANIPULATIONS' drawer
will be created for the demo.(This archiwe contains already installed
version.You can run it by clicking on icon(if You have more then 2Mb Ram)
or by typing 'Manipulations.exe' fro CLI.)
If You have only 2Mb of RAM please boot with no startup and disable
unnesesary partition).
This demo take about nine month of our hard work,so don't take any profit
of it.Please spread it wherewer You can.
