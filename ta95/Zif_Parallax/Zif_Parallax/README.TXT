
How to install ZiF?
~~~~~~~~~~~~~~~~~~~
		A. Make a destination directory called ZiF to your HD.
					( MAKEDIR dh0:ZiF )

		B. Copy all files from disk(s) to destination directory!
					( COPY df0:#? dh0:ZiF )

		C. Voila

How to start THE Demo?
~~~~~~~~~~~~~===~~~~~~

		A. Go to ZiF directory.
					( dh0:ZiF )

		B. Type 'ZiF.run' and press return.

		C. Watch a brilliant demo?
