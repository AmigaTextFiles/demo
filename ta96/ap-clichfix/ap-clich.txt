
ARTIFICIAL PEOPLE: CLICHES
~~~~~~~~~~~~~~~~~~~~~~~~~~
Our first demo as a contribution to the Assembly'96 Amiga-democompo.
Don't forget to run SetPatch and enable ALL possible caches before
running this demo! (ie. run CPU CACHE BURST) Doesn't like fragmented
memory so you'd better run this right after a clean boot...

Code, graphics, end tune	- Kalsu
Object				- Iodine
Raytracing			- Thain/Sensible
Main tune			- Falcon
Moral support			- Cob

Requires A1200 + 3MB Fast (68030 recommended)

Tested on A1200 + 030/50 + 8MB Fast
          A1200 + 060/50 + 16MB Fast
          A1200 + 4MB Fast
